# Driver Bot — этап 1: базовый бот + Google Sheets

Это первый рабочий кусок проекта: бот подключается к Telegram и читает данные
из Google-таблицы (водитель → пломбы).

## Что уже работает

- `/start` — приветствие и список команд
- `/myseals` — бот ищет в таблице строку по твоему имени из Telegram и
  показывает закреплённые за тобой номера пломб

## Шаг 1. Создать бота в Telegram

1. Напиши в Telegram **@BotFather**
2. Команда `/newbot`, придумай имя и юзернейм
3. Получишь токен вида `123456789:ABCdefGhIJKlmNoPQRsTuVwXyZ` — он понадобится дальше

## Шаг 2. Настроить Google Service Account (доступ к таблице)

1. Зайди в [Google Cloud Console](https://console.cloud.google.com/)
2. Создай новый проект (или используй существующий)
3. В меню «APIs & Services» → «Library» включи:
   - **Google Sheets API**
   - **Google Drive API**
4. В «APIs & Services» → «Credentials» → «Create Credentials» → **Service Account**
   - Дай любое имя, роль можно не назначать
5. Зайди в созданный Service Account → вкладка «Keys» → «Add Key» → **JSON**
   — скачается файл ключа
6. Переименуй скачанный файл в `google_service_account.json` и положи в
   папку `credentials/` в корне проекта
7. Открой свою Google-таблицу → «Настройки доступа» → дай доступ
   **email сервисного аккаунта** (он выглядит как
   `имя@проект.iam.gserviceaccount.com`, найдёшь его в JSON-файле или в
   консоли) — права **Редактор**

## Шаг 3. Подготовить таблицу

В таблице должна быть первая строка с заголовками (именно так, на русском):

| Водитель | Контейнер | Пломбы | Статус |
|---|---|---|---|
| Иванов Иван Иванович | CONT1234567 | 12345, 67890 | |

ID таблицы — это часть ссылки между `/d/` и `/edit`:
`https://docs.google.com/spreadsheets/d/ВОТ_ЭТО_ID/edit`

## Шаг 4. Установить зависимости

```bash
python -m venv .venv
source .venv/bin/activate   # на Windows: .venv\Scripts\activate
pip install -r requirements.txt
```

## Шаг 5. Настроить переменные окружения

Скопируй `.env.example` в `.env` и заполни:

```bash
cp .env.example .env
```

Впиши туда токен бота и ID таблицы.

## Шаг 6. Запустить

```bash
python -m bot.main
```

Если всё настроено верно — бот ответит на `/start` в Telegram, а `/myseals`
покажет пломбы из таблицы (имя в Telegram должно совпадать со столбцом
«Водитель» — это временное решение для теста, на следующем этапе сделаем
нормальную привязку через регистрацию).
